Documentation de configuration

Vérification de TLS 1.2 dans iNot Comptabilité

Vérifiez bien dans le logiciel iNot Comptabilité, dans les paramètres généraux, que TLS 1.2 est bien activé.

Vérification de TLS 1.2 dans Internet Explorer

Vérifiez bien que TLS 1.2 est activé dans les paramètres avancés d'Internet Explorer.

Vérification de l'homologation dans iNot Comptabilité

Vérifiez bien que vous êtes en homologation dans les paramètres WebServices du logiciel iNot Comptabilité.

Vérification des URLs dans iNot Comptabilité

Vérifiez bien les URLs dans les configurations WebServices du logiciel iNot Comptabilité.

Vérification du CRPCEN dans iNot Comptabilité

Vérifiez bien le CRPCEN dans les configurations du logiciel iNot Comptabilité (ctrl+shift+D), il doit être du même numéro que la carte.
